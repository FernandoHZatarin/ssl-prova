# Exercício 8 - Servidor HTTPS com certificado real

## Comandos utilizados
python3 /etc/ssl/certificates/servidor.py

## Explicação
Agora com Script certo do Python usamos o aluno.key para subir um servidor HTTPS de verdade. O navegador mostrou o aviso de segurança porque é certificado por autoassinado, mas a conexão estava de fato criptografada.
